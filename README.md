# Stock price predict system
Easy lstm model and finance data preprocessing

時系列モデル（Lstm）を用いた株価予測システム

過去一週間分のデータを元に翌日の株価が上がるか予想する。

※このコードは実運用しているシステムの一部を抜粋し、簡易化したものです。
